O que é um site

Introdução ao HTML

  • É a sigla para HyperText Markup Language
  • "Esqueleto" de qualquer página da web
  • Definição na MDN:

    "HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web [...] 'Hipertexto' refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web."

  • Exemplo:

    • página da DH Schools completa / página da DH Schools apenas com o HTML

Mas como é um código HTML?

Esta é a estrutura básica que você irá encontrar em todos os documentos HTML:

<!DOCTYPE html>

<html>

<head>

<title>

</head>

<body>

<h1>

</h1>

<p>

</p>

</body>

</html>

Como desenvolver código?

  • Do que você precisa (por enquanto): um computador e um bloco de notas
  • Para facilitar a vida, tenha um editor de texto: Atom, VSCode, Sublime Text ou outros
  • Extensões: para deixar do jeito que você gosta

Mão na massa

  • Abra o bloco de notas e salve o arquivo com o nome index.htlm em uma pasta na sua máquina
  • Escreva a estrutura básica de uma página web que aprendemos com um conteúdo qualquer
  • Agora abra um novo arquivo com o nome style.css e mude a cor de fundo para red
  • No arquivo index.html, faça o link para o seu arquivo css
  • Agora abra o arquivo index.html no seu navegador
  • Você criou a sua primeira página! :)
