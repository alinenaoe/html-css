Introdução ao HTML

  • É a sigla para HyperText Markup Language
  • "Esqueleto" de qualquer página da web
  • Definição na MDN:

    "HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web [...] 'Hipertexto' refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web."

  • Exemplo:

    página da DH Schools completa / página da DH Schools apenas com o HTML

Mas como é um código HTML?

Esta é a estrutura básica que você irá encontrar em todos os documentos HTML:

<!DOCTYPE html>

<html>

<head>

<title>Título da sua página na aba do navegador</title>

</head>

<body>

<h1> Sou um título</h1>

<p>Sou um parágrafo</p>

</body>

</html>

Características importantes do HTML

  • Hierarquia: percebemos a relação entre os elementos por meio de uma indentação correta
  • Semântica: usando tags corretas não só estruturamos código, mas damos significado a ele
  • Exemplos de tags: header, nav, footer

Estrutura de uma tag

<section class="tema" id="estrutura-tag">

Aqui vai o conteúdo da sua seção. Class é um atributo do elemento section, serve para classificar elementos que tenham algo em comum, por exemplo um estilo ou comportamento. Ou seja, várias seções podem ter a mesma class

Já o atributo id foi feito para ser atribuído a apenas um elemento em todo o seu código: ele deve ser único

</section>

Introdução ao CSS

  • É a sigla para Cascading Style Sheets
  • Uma linguagem de folha de estilos que descreve como o os elementos HTML devem se comportar na tela
  • Uma boa prática é criar um arquivo separado do arquivo html, para melhor organização e manutenção do código

Mas como é um código CSS?

Para estilizar um HTML, é preciso "chamá-los" pelo CSS e dizer quais são as propriedades:

body {

background-color:red

}

.nome-da-classe {

font-family: 'Lato', sans-serif;

}

#nome-da-ID {

margin-top: 20px

}

O que dá para fazer com CSS?

  • Posicionar elementos
  • Indicar estilo de fonte, cores e espaçamento entre elementos
  • Mudar o comportamento de um elemento ao passar o mouse, por exemplo
  • Restringir o tamanho de um bloco ou imagem
  • Adaptar sua página para a visualização em dispositivos móveis

Mão na massa

  • Abra o bloco de notas e salve o arquivo com o nome index.htlm em uma pasta na sua máquina
  • Escreva a estrutura básica de uma página web que aprendemos com um conteúdo qualquer
  • Agora abra um novo arquivo com o nome style.css e mude a cor de fundo para red
  • No arquivo index.html, faça o link para o seu arquivo css
  • Agora abra o arquivo index.html no seu navegador
  • Você criou a sua primeira página! :)
